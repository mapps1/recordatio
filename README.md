# Recordatio

Recordatio is a TypeScript + Express wellness tracking app.

## Commands

- `npm install`
- `npm run dev`
- `npm test`
- `npm run build`
- `npm start`

## API

- `GET /api/health`
- `GET/POST/DELETE /api/checkins`
- `GET/POST/PUT/DELETE /api/journaling`
- `GET /api/dashboard`
- `GET /api/resources`
- `GET /api/alerts`
